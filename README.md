# QR Code Generator
Всем привет я создал просто и удобный скрипт для генерации QR-кодов из текста или ссылки
- Генерация QR-кодов из произвольной строки
- Поддержка настройки размера ячеек
- Настраиваемая ширина границы
- Указание имени выходного файла
- Работает как CLI-утилита

Импорты:
все что нужно это pip install Pillow
